## 🧾 About server

This is an open server designed to work with MAX systems. It helps you connect and communicate easily using the MAX API and MAX Messenger tools. 

You do not need to know any programming to use this software. Just follow the steps here to get it running on a Windows computer.

---

## 📋 System Requirements

- Operating System: Windows 10 or later  
- CPU: 2 GHz or faster processor  
- RAM: 4 GB minimum  
- Storage: At least 500 MB free space  
- Network: Internet connection recommended for updates and messaging features

## ▶️ Running the server

1. **Start the server application:**  
   After installation, find the server app icon on your desktop or in your Start menu.

2. **Launch the app:**  
   Double-click the icon to open the server.

3. **Initial setup window:**  
   The server will start and show the main control panel. Here you can check connection status and settings.

4. **Connect to MAX API:**  
   Click "Connect" to link the server to your MAX API account. If you do not have login details, contact your MAX system administrator.

  Add or remove users who can connect through MAX Messenger.

## 📡 Using MAX Messenger

The server lets you send and receive messages through MAX Messenger. Messages pass through this server to stay inside your network.

- Open your MAX Messenger app on your device.  
- Make sure it is connected to the server IP address shown in the server control panel.  
- Use the normal messaging features to chat with other users on your network.

---

## 🗂️ File Storage and Logs

- The server saves logs and message backups in a default folder:  
  `C:\ProgramData\server\logs`  

- You can view logs for troubleshooting or archive old messages here.

- Log files are text-based and can be opened with Notepad.

| Server won’t start          | Restart your computer and try again. Check Windows security prompts.  |
| Cannot connect to MAX API  | Check your network and firewall settings. Confirm API credentials.   |
| Messages do not send       | Verify network connection. Restart the server if needed.             |
| Installer fails            | Run installer as administrator (right-click, Run as administrator).  |
